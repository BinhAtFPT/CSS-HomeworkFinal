Linux Distro Recommendations

Ubuntu

Ubuntu is a distro made by Canonical with multiple editions for Server, IoT or normal Desktop. It is beginner-friendly, stable as it is made for Enterprise and has technical support if you ever need them.

Linux Mint

Linux Mint is also a beginner-friendly distro based on Ubuntu LTS with an Interface that gives users a similiar feelings to Windows. Since it based on Ubuntu it comes with a few advantage that Ubuntu have along with a longer support times with Long Term Support Release.

Fedora

Fedora has similiar design goals to Ubuntu. However, instead of base on Debian like Ubuntu, Fedora is actually an upstream version of Red Hat Enterprise Linux and is used by Linus Torvalds, creator of the Linux Kernel

Pop! OS

Pop! OS is also based on Ubuntu with better support for NVIDIA GPU, streamlined window and workflow management alongside few Quality-of-Life improvements. It is made by System76, a Linux Computer manufacturer.

Arch Linux

Arch Linux is actually isn't a beginner-friendly Linux distro. It actually is a Do-it-yourself Linux distro which isn't beginner-friendly at all. However, if you wanted to learn more about Linux, you should give this a try.
